# Talent Shopping Platform

A comprehensive talent management and agency shopping platform built with React and Firebase.

## Features

- Model management with photos and portfolios
- 350+ agencies across 29+ territories
- Project/campaign tracking with status updates
- Agency signing history tracking
- Batch email generation with templates
- Real-time sync across devices via Firebase

## Technology Stack

- React 18
- Firebase Firestore
- Tailwind CSS
- Single-file HTML application

## Deployment

Hosted on GitHub Pages with Firebase backend.
